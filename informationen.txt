Gibbsta App - Projektplan
🎯 Projektziel
Eine Social Media App für die GIBB-Schule mit Instagram-ähnlichen Features: Bilder hochladen, kommentieren und chatten.
📋 Kernfunktionen
1. Benutzer-Management

Registrierung/Login mit Schul-E-Mail Verifizierung
Profilerstellung (Name, Klasse, Profilbild)
Profil bearbeiten und anzeigen

2. Bild-Features

Bilder hochladen (mit Kamera oder Galerie)
Filter/Bearbeitung (optional für v1.0)
Bildergalerie (Feed-Ansicht)
Likes für Bilder

3. Interaktion

Kommentarsystem unter Bildern
Chat-Funktion (1-zu-1 und Gruppenchats)
Benachrichtigungen für Likes, Kommentare, Nachrichten

4. Schul-spezifische Features

Klassenzugehörigkeit anzeigen
Schulevents posten/teilen
Fächer-Tags für Posts

🛠️ Technologie-Stack (Empfehlung)
Frontend (Mobile App)

React Native oder Flutter

✅ Eine Codebasis für iOS und Android
✅ Große Community und viele Tutorials
✅ Gut für Anfänger geeignet



Backend

Node.js mit Express.js

✅ JavaScript überall (Frontend + Backend)
✅ Viele verfügbare Pakete


Alternative: Firebase (No-Code Backend)

✅ Einfacher Einstieg
✅ Integrierte Authentifizierung
✅ Realtime Database



Datenbank

MongoDB (mit Node.js) oder Firestore (mit Firebase)
Cloudinary für Bildspeicherung

Tools & Services

Git/GitHub für Versionskontrolle
VS Code als Editor
Postman für API-Tests

📅 Zeitplan (8-12 Wochen)
Phase 1: Planung & Setup (Woche 1-2)

 Detaillierte Anforderungsanalyse
 UI/UX Mockups erstellen (Figma)
 Entwicklungsumgebung einrichten
 Git Repository aufsetzen

Phase 2: Grundfunktionen (Woche 3-6)

 Benutzer-Registrierung/Login
 Basis-Profilerstellung
 Einfacher Bild-Upload
 Basis-Feed implementieren

Phase 3: Erweiterte Features (Woche 7-10)

 Kommentarsystem
 Chat-Funktion
 Benachrichtigungen
 Like-System

Phase 4: Testing & Polish (Woche 11-12)

 Bug-Fixes
 Performance-Optimierung
 Beta-Testing mit Mitschülern
 App Store Vorbereitung

🚀 MVP (Minimum Viable Product)

✅ Bilder hochladen
✅ Bilder anzeigen (Feed)
✅ Einfache Kommentare
